# NGO Management System

A comprehensive Flask-based web application for managing NGO operations, donors, beneficiaries, volunteers, and events.

## Features
- Multi-NGO support
- Donor management & tracking
- Volunteer coordination with skill matching
- Event management with budget tracking
- Fund redistribution system
- Automated data validation triggers

## Tech Stack
- Backend: Python Flask
- Database: MySQL 8.0
- Frontend: HTML5, CSS3, JavaScript, Jinja2

## Setup
1. Install requirements: `pip install -r requirements.txt`
2. Configure database in `config.py`
3. Run: `python app.py`
